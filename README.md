# 🌸 知识花园

一个为小学 1-6 年级学生设计的全科答题经营游戏。通过答对数学、语文、英语题目赚取「阳光币」和「星星」，建造属于自己的梦幻花园。

🌐 **在线游玩：** https://chacelee.github.io/magic-garden/

---

## 🎮 怎么玩

### 📝 答题（三学科切换）
- 顶部切换 **🔢 数学 / 📝 语文 / 🔤 英语**
- 选择 **年级 → 学期 → 单元**，点 **「开始答题」**
- 每道题 4 个选项，答对获得 **☀️ 阳光币** 和 **EXP 经验值**
- **🔥 Combo 连击** — 连续答对奖励翻倍
- 答错不扣分，显示正确答案，自动加入复习队列

### 🧠 智能复习
- 基于 **SM-2 算法**（Anki 同款）自动安排复习
- 答对：间隔递增（1天→3天→7天→15天→30天）
- 答错：间隔重置为 1 天
- 答题页顶部显示 **「📅 今日复习」** 入口

### 🌻 花园
- 商店购买种子、树木、装饰品和宠物
- 点击 **「编辑」** 进入布置模式，从背包拖放到网格
- 植物随时间生长：🌱 → 🌿 → 🌻
- **🗺️ 扩展** — 达到等级可花钱扩大花园，价格逐次上涨 10%

### ⭐ 星星 & 宠物
- 5 连击可获得星星，用来购买宠物（小猫、小兔、小鸟等）

### 🏆 成就
- 18 个成就徽章，覆盖答题/连击/花园/签到
- 每日签到领奖励

## 📚 课程内容

全部 **3 学科 × 6 年级 × 2 学期 = 36 个学期** 的内容：

| 学科 | 覆盖范围 |
|------|---------|
| 🔢 **数学** | 1-6年级人教版：加减乘除→分数小数→方程→几何→比例 |
| 📝 **语文** | 拼音、识字、组词、古诗（咏鹅→石灰吟）、成语、修辞、文言文 |
| 🔤 **英语** | 字母→单词→句型→时态（人教PEP 1-6年级） |

## 📊 家长面板

- 完整学习报告（按学科查看正确率和进度）
- 错题集（按学科分组）
- 设置：**12档时间按钮**（5s~120s）、音效开关、每日学习时长

## ☁️ 云存档（跨设备同步）

基于 Firebase Cloud Firestore 实现跨设备进度同步：

1. 在设备 A 点 **「💾 保存进度」** → 生成 6 位存档码
2. 在设备 B 输入同一存档码 → **「📥 从云端恢复」**
3. 等级、金币、花园、学习进度全部恢复

## 💡 设计理念

- **无惩罚** — 答错不扣分，鼓励尝试
- **自适应难度** — 根据正确率自动调整
- **间隔复习** — 科学算法巩固记忆
- **经营激励** — 用花园建造驱动学习

## 🛠️ 技术

纯前端 HTML + CSS + JavaScript + Firebase。

| 依赖 | 用途 |
|------|------|
| Firebase Firestore | 云存档同步 |
| Web Audio API | 音效（零外部文件） |
| localStorage | 本地数据持久化 |
