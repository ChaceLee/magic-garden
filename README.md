# 🌸 知识花园

一个为小学 1-6 年级学生设计的全科答题经营游戏。通过答对数学、语文、英语题目赚取「阳光币」和「星星」，建造属于自己的梦幻花园。花园部分采用 **Phaser 3** 引擎渲染，带来流畅的 2D 交互体验。

🌐 **在线游玩：** https://chacelee.github.io/magic-garden/

---

## 🎮 怎么玩

### 🌻 花园（Phaser 引擎）
- 花园采用 **Phaser 3** 2D 引擎渲染，支持流畅动画和交互
- 植物生长过程可视化：幼苗 → 成长 → 盛开，伴随摇摆动画
- 宠物（小猫、小兔、小鸟等）有跳跃动画
- 房子可升级：茅草屋 → 木屋 → 砖房 → 石头房 → 别墅
- 编辑模式下点击网格放置/移除物品

### 📝 答题（三学科切换）
- 顶部切换 **🔢 数学 / 📝 语文 / 🔜 英语**
- 选择 **年级 → 学期 → 单元**，点 **「开始答题」**
- 每道题 4 个选项，答对获得 **☀️ 阳光币** 和 **EXP 经验值**
- **🔥 Combo 连击** — 连续答对奖励翻倍
- 答错不扣分，显示正确答案，自动加入复习队列

### 🧠 智能复习
- 基于 **SM-2 算法**（Anki 同款）自动安排复习
- 答对：间隔递增（1天→3天→7天→15天→30天）
- 答错：间隔重置为 1 天
- 答题页顶部显示 **「📅 今日复习」** 入口
- 复习模式新增音效反馈（答对/答错均有提示音）

### ☁️ 云存档（跨设备同步）

基于 Firebase Cloud Firestore 实现跨设备进度同步：

1. 在设备 A 点 **「💎 保存进度」** → 生成 6 位存档码
2. 在设备 B 输入同一存档码 → **「📥 从云端恢复」**
3. 等级、金币、花园、学习进度全部恢复

### ⭐ 星星 & 宠物
- 5 连击可获得星星，用来购买宠物（小猫、小兔、小鸟等）

### 🏆 成就
- 18 个成就徽章，覆盖答题/连击/花园/签到
- 每日签到领奖励

## 📚 课程内容

全部 **3 学科 × 6 年级 × 2 学期 = 36 个学期** 的内容：

| 学科 | 覆盖范围 |
|------|---------|
| 🔢 **数学** | 1-6年级人教版：加减乘除→分数小数→方程→几何→比例 |
| 📝 **语文** | 拼音、识字、组词、古诗（咏鹅→石灰吟）、成语、修辞、文言文 |
| 🔜 **英语** | 字母→单词→句型→时态（人教PEP 1-6年级） |

## 📊 家长面板

- 完整学习报告（按学科查看正确率和进度）
- 错题集（按学科分组）
- 设置：**12档时间按钮**（5s~120s）、音效开关、每日学习时长

## 💡 设计理念

- **无惩罚** — 答错不扣分，鼓励尝试
- **自适应难度** — 根据正确率自动调整
- **间隔复习** — 科学算法巩固记忆
- **经营激励** — 用花园建造驱动学习

## ⚙️ 技术栈

纯前端 HTML + CSS + JavaScript + Phaser + Firebase。

| 特性 | 技术方案 |
|------|---------|
| **花园引擎** | Phaser 3（CDN），程序化纹理生成，tween 动画 |
| **UI 主题** | 明亮高对比色彩体系，CSS 变量全局控制 |
| **数据持久化** | localStorage（防抖写入）+ Firebase Cloud Firestore |
| **音效** | Web Audio API（零外部文件） |
| **复习算法** | SM-2 间隔重复（Anki 同款） |
| **动画效果** | Phaser tween + CSS keyframes |

| 依赖 | 用途 |
|------|------|
| Phaser 3 | 花园场景 2D 引擎渲染 |
| Firebase Firestore | 云存档同步 |
| Web Audio API | 音效（零外部文件） |
| localStorage | 本地数据持久化 |

## ⚡ 主要优化

- **Debounced 保存**：高频状态变更合并为一次写入，大幅减少磁盘 I/O
- **懒加载题目数据**：语文/英语题库按需动态加载，减少 102KB 初始下载体积
- **AudioContext 手势激活**：修复 Chrome/iOS 首次播放音效失败问题
- **植物生长通知**：进入花园时自动提示新长成的植物
