# 🌸 梦幻数学花园

一个为小学1-2年级学生设计的数学答题经营游戏。通过答对数学题赚取「阳光币」和「星星」，用来打造属于自己的梦幻花园。

## 🚀 快速开始

1. **直接打开** — 双击 `index.html` 在浏览器中打开即可游玩
2. **或用服务器启动**（推荐，体验更好）：
   ```
   npx serve . -p 3000
   ```
   然后在浏览器打开 `http://localhost:3000`

## 🎮 怎么玩

### 📝 答题
1. 在「答题」页面选择 **年级 → 学期 → 单元**
2. 点击 **「开始答题」**
3. 每道题有4个选项，选对了获得 **☀️ 阳光币** 和 **EXP经验值**
4. 连续答对会触发 **🔥 Combo连击**，奖励翻倍！
5. 答错不扣分，会显示正确答案，下次会自动复习错题

### 🌻 花园
- 答对的阳光币可以在 **商店** 购买种子、树木、装饰品和宠物
- 进入花园后点击 **「编辑」**，从背包选择物品，再点击网格放置
- 植物会随时间生长：🌱 种子 → 🌿 生长中 → 🌻 盛开

### ⭐ 星星
- 答对 **5连击** 可以获得星星
- 星星用来购买 **宠物**（小猫、小兔、小鸟等）

### 🏆 成就
- 答对一定数量题目、达到连击、连续签到等都会解锁成就徽章

### 📊 家长面板
- 在「成就」页下方可以看到完整的学习报告
- 查看每个知识点的正确率、错题集、学习进度
- 可以设置倒计时音效、题目时间、每日学习时长

## 📚 课程内容

| 学期 | 知识点 |
|------|--------|
| 一年级上 | 1~5/6~10/11~20 加减法 → 认识钟表 → 认识图形 |
| 一年级下 | 20以内退位减法 → 100以内数 → 认识人民币 → 找规律 |
| 二年级上 | 长度单位 → 100以内加减(二) → 认识角 → 乘法口诀(2~9) |
| 二年级下 | 混合运算 → 万以内数 → 克与千克 → 有余数除法 → 数据整理 |

## 💡 设计理念

- **无惩罚** — 答错不扣分，显示正确答案即可
- **自适应难度** — 正确率高自动升级到挑战题
- **错题闭环** — 自动收集错题，优先复习
- **经营激励** — 用游戏奖励驱动学习，让学习变得有趣
- **离线可用** — 纯前端，所有数据存浏览器，断网也能玩

## 🛠️ 技术

纯 HTML + CSS + JavaScript，零依赖，任何浏览器都能运行。
