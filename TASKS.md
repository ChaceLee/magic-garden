# 🎮 知识花园 — 扩展任务计划

> - ✅ 自适应学习路径
> - ✅ 花园深度开发（粒子特效 + 天气/昼夜循环）

---

## 🎯 剩余扩展方向

### 🔶 游戏化机制（中优先级）

1. **多人在线花园拜访**
   - 基于 Firebase 实现拜访好友花园（只读）
   - 花园点赞/短评
   - 排行榜：“本周答题之星”“最强连击王”“花园设计师”

2. **迷你花园小游戏**
   - 植物知识问答：点击已长成的植物，弹出科普小知识
   - 花园谜题：每天随机出现布局谜题，完成奖励种子
   - 宠物喂养问答：宠物饿了出题，答对才能喂食

### 🔷 学习机制增强（中优先级）

3. **错题本增强 → 智能组卷**
   - 错题按知识点自动归类
   - 一键生成针对性练习题
   - 错题清零弹出庆祝动画 + 成就徽章

4. **学习计时与专注力管理**
   - 答题界面内置番茄钟
   - 达到学习时长后弹出休息提醒
   - 每日学习报告（已学时间、题量、正确率趋势图）

### 🔸 技术扩展（低优先级）

5. **Phaser 花园深度开发续**
   - 人物 Avatar：学生可自定义角色，在花园中走动
   - 花园 BGM：用 Web Audio API 生成简单的背景音乐

6. **学科扩展**
   - 🔬 科学（1-6年级）：动植物、物理现象、地球宇宙
   - 🎨 美术：色彩认知、名画赏析、基础构图

7. **PWA 支持**
   - 添加 Service Worker，支持离线答题
   - 可安装到手机桌面
   - 推送通知：每日复习提醒、植物长成通知
